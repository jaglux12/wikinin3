Primeros pasos - VSCode

Visual Studio Code es un editor de código desarrollado por Microsoft para sistemas operativos Windows. Con una mirada en el desarrollo de proyectos abiertos, Microsoft decidió compartir con todo el público el código fuente del programa. De esta forma, se permite que VSCode esté disponible en varios sistemas operativos, como Mac y GNU/Linux.
